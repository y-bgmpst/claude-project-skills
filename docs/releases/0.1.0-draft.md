# Claude Project Skills v0.1.0 Draft

Status: draft

## Scope

This first release captures the repository bootstrapping work:

- reusable skill packaging under `skills/`
- local install helper for `~/.claude/skills`
- repo-level validation
- GitHub Actions validation on PRs and `main`
- in-depth wiki documentation
- a starter set of reusable Claude skills
- a token-lean router skill and context groups

## Validation

Recommended checks before publishing the release:

```bash
python3 scripts/validate_skill_repo.py
./scripts/install_to_claude_home.sh
```

## Release notes

This release is intentionally small and focused.

It establishes the source-of-truth repository for reusable Claude skills and a predictable workflow for installing, reviewing, and maintaining them.

## Follow-up candidates

- split helper scripts into a dedicated tooling subdirectory if they grow further
- add more project-specific skills when they earn their keep
- expand the wiki with examples or troubleshooting notes only when those are repeatedly needed
