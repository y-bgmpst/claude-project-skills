# Container Ops

Use this group for Kubernetes or Docker tasks.

## Route

- Kubernetes manifests, kubectl, namespaces, RBAC, rollouts, or cluster access -> `skills/kubernetes-docker-ops-guardrails/Skill.md`
- Dockerfiles, images, Compose, container hardening, or runtime debugging -> `skills/kubernetes-docker-ops-guardrails/Skill.md`

## Minimal load order

1. Read the target object or file first.
2. Keep the cluster context or image tag explicit.
3. Mutate only after the scope is narrow.
4. Verify the result immediately after the change.
