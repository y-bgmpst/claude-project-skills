# Common Tasks

Use this group when choosing which skill to suggest for a normal user task.

## Suggested skill routing

- email drafting or rewriting -> `email-draft-polish`
- meeting notes or transcripts -> `meeting-notes-and-actions`
- spreadsheet formulas or pivots -> `spreadsheet-formula-helper`
- file cleanup or folder organization -> `file-organizer`
- repo skill selection or bootstrap -> `bootstrap-router`

## Principle

If the task is not one of the above, avoid guessing. Load the repo-specific group instead.
